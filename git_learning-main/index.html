GIT LEARNING

Git est un VCS Version Control System(Système de controle de version).untracked est eu ensemble de fichiers non reconnu par github. Nous disposons d'un système

  • local
  • central
  • distributive

Git fonctionne avec des fichiers instannés.

Nous disposons de trois espaces de travail:

  • L'espace de travail
  • La Zone d'index
  • Le Répertoire GIT

git init: initialise le terminal

cls/clear/CTRL+L: efface le terminal

dir liste le contenu du dossier ouvert

cd change le chemin d'accès pour l'ouverture d'un dossier/changer de Répertoire

git config --global user.name pour configurer le nom d'utilisateur

git config h pour avoir de l'aide depuis le système

git config --list pour lister tous les éléments disponible dans le terminal d'une part et d'autre part, config local juste pour configurer un fichier local

pour indexer un fichier en attente on ulise le git add avec le nom du fichier

git status permet de voir l'état

git commit permet de valider les modifications effectuées. On fait git commit -m "le message "

git log permet d'avoir les informations par rapport a ce qui a été enregistrer

:wq=w représente sauvegarder et q reprsente quit pour sortir du mode insertion.

code . pour ouvrir et le vsc et le dossier.
